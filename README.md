# jsESlanguage
Tools for process Spanish language

### jsESinflection: 
  Spanish inflections

### jsESsyllable:
Spanish syllables division

### jsEStemmer:
Stemmer for spanish. Not only stem words too remove stop words, group similar lemmas and filter results.
